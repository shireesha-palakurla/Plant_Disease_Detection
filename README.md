🌿 Plant Disease Detection using Deep Learning

# Project Title

AI-Based Plant Disease Classification System using CNN.
Model Accuracy: 93.16%

# 🧠 Project Description

This project is an image-based plant disease detection system that uses Deep Learning (Convolutional Neural Networks - CNN) to automatically identify diseases in plant leaves. The model is trained on the popular plant disease dataset available on Kaggle, which contains thousands of labeled images of healthy and diseased crop leaves.

The system focuses on major crops such as Tomato, Potato, and Pepper, and classifies leaf images into different disease categories like early blight, late blight, bacterial spots, and healthy leaves.

# 🎯 Objective

The main objective of this project is to:
Help farmers and agricultural experts detect plant diseases early
Reduce crop loss by providing fast and accurate predictions
Automate disease identification using AI instead of manual inspection

# ⚙️ How It Works
User uploads an image of a plant leaf;
Image is preprocessed (resized, normalized).
CNN model analyzes patterns like spots, color changes, and textures.
The trained model predicts the disease class with probability score.
Output shows whether the plant is healthy or infected and the disease name.
